# Changelog

## 0.1.0 (Initial Release)

- Config flow with URL, API key, tenant selection
- 4 coordinators (plant, location, alert, task)
- Sensors for plants, runs, locations, tanks
- Binary sensors (attention, care, sensor status)
- Calendar entities (phases, tasks)
- Todo entity with backend sync
- 5 services (fill_tank, water_channel, confirm_care, refresh_data, clear_cache)
- 5 custom Lovelace cards (plant, mix, tank, care, houseplant)
- 11 event types
- DE/EN translations
- Diagnostics
